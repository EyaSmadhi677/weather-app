# Weather App

A beautiful, responsive weather application that displays current weather information for any city using the OpenWeatherMap API.

## Features

- Real-time weather data
- Beautiful glass-morphism UI
- Responsive design
- Search functionality
- Weather icons matching conditions

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## Setup

1. Clone this repository
2. Open `index.html` in your browser
3. Enter a city name to see the weather

## API Key

This app uses OpenWeatherMap API. You can get your own free API key from https://openweathermap.org/api
